AWS Credentials

You can create AWS credentials in the AWS IAM Console . Make sure to grant S3 permissions to your bucket.

S3-Compatible Services

This addon works with S3-compatible services like MinIO, DigitalOcean Spaces, Wasabi, and others. Use the custom endpoint URL field for these services.

Storage Classes
  • Standard: For frequently accessed data
  • Standard-IA: For infrequently accessed data
  • Glacier: Long-term archival (retrieval time: minutes to hours)
  • Deep Archive: Lowest cost (retrieval time: 12+ hours)
